工欲善其事必先利其器，在进行 Angular 的开发时，通常会结合 Webh5ack 和 ES6 一起进行。
本项目讲的正是如何使用 AngularJs + Webpack + ES6 进行组合开发。
本项目所涉及的代码 ，可以直接作为一个种子项目使用。
项目内置了两个开发中比较常用的模块 用户 “登录验证” 以及 “路由鉴权” 功能。
快速构建

快捷的构建AngularJs应用程序, 使用npm管理所有开发依赖.

git clone https://github.com/16code/AngularJs-uiRouter-Authorize-Demo.git
npm install -g webpack eslint
cd AngularJs-uiRouter-Authorize-Demo
npm install
npm start
组件式开发

组件式开发, 最后将组件组合成页面, 使用ocLazyload轻松实现按需加载.

import myComponent from '../../components/myComponent';
const module = angular.module('app.moduleName', [
    myComponent.name
]);
export default module.name;
